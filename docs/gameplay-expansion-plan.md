# 플레이타임 확장안

## 현재 구조 진단

- 현재 씬 수는 `15개`, 총 선택지는 `30개`다.
- 초반 5개 씬 이후 분기가 빠르게 합쳐져서, 실제 체감 플레이타임은 짧다.
- `record_scene`, `citizen_voice`, `community`처럼 큰 허브 씬으로 너무 빨리 수렴한다.
- 현재 분기는 "한 번 선택하고 바로 다음 장면" 구조라서, 긴장 축적이나 후속 결과를 체감하기 어렵다.
- 스탯은 쌓이지만 분기 해금이나 결과 차이로 거의 돌아오지 않아 게임성이 약하다.

## 바로 늘릴 수 있는 방향

### 1. 허브 씬 앞에 중간 씬을 끼운다

가장 효과가 큰 방법이다. 지금 구조는 아래 씬들이 너무 빨리 합류한다.

- `start` -> `university_gate`
- `university_gate` -> `record_scene` / `downtown`
- `radio_room` -> `record_scene` / `citizen_voice`
- `help_people` -> `community`
- `community` -> `last_night` / `archive_ending`

이 사이에 1~2개씩 중간 씬을 넣으면 플레이타임이 자연스럽게 늘어난다.

### 2. 선택의 결과를 1단계 더 보여 준다

지금은 선택 = 바로 다음 장소 이동이다.

아래처럼 바꾸면 체감이 좋아진다.

- "전남대 쪽으로 간다" -> "검문을 피해 골목으로 돈다" -> "정문 앞 도착"
- "이웃에게 설명한다" -> "믿지 못하는 이웃과 대화" -> "도청 앞으로 함께 감"
- "기록한다" -> "증언 한 명을 더 붙잡는다" -> "정리된 기록 확보"

### 3. 스탯이 실제 분기를 열게 만든다

현재 스탯은 표시만 된다. 아래처럼 쓰면 게임성이 생긴다.

- `record >= 2`면 "증언을 묶어 정리한다" 선택 해금
- `trust >= 2`면 "주민들이 먼저 문을 열어 준다" 선택 해금
- `safety >= 2`면 "우회로로 이동한다" 선택 해금
- `courage >= 2`면 "도청에 남는다" 선택이 더 설득력 있게 열림

### 4. 소지품을 분기 장치로 쓴다

현재 인벤토리는 장식에 가깝다. 실제 분기에 연결하면 좋다.

- `수첩`이 있으면 기록 분기 강화
- `물통`이 있으면 부상자 지원 분기 강화
- `연락처 메모`가 있으면 외부 전달 분기 강화
- `전단/소식지`가 있으면 시민 설득 분기 강화

## 추천 확장 구조

현재 15개 씬을 24~28개 정도로 늘리는 구성이 적당하다.

### A. 초반 확장

#### 1. `station_rumor`

- 위치: `start`와 `university_gate` 사이
- 역할: 광주역 앞 시민들의 말, 소문, 긴장 확인
- 효과: 첫 분기 체감 증가, 분위기 축적
- 기존 배경 재사용 가능: `station`

#### 2. `side_alley_detour`

- 위치: `university_gate` 진입 전
- 역할: 정문 대신 골목으로 우회하며 검문 분위기 체감
- 효과: 이동 자체가 선택이 됨
- 신규 배경 있으면 좋음

#### 3. `family_neighborhood`

- 위치: `call_family` 이후
- 역할: 가족 통화 후 집 근처 이웃과 짧은 대화
- 효과: 사적 불안과 공적 상황 연결
- 기존 배경 재사용 가능: `phonebooth` 또는 `home`

### B. 중반 확장

#### 4. `street_clinic`

- 위치: `record_scene`와 `help_people` 사이
- 역할: 임시 치료, 물 나르기, 사람 연결
- 효과: 돌봄과 공동체 체감 강화
- 신규 배경 추천

#### 5. `leaflet_room`

- 위치: `radio_room` 또는 `record_scene` 이후
- 역할: 들은 사실을 정리하거나 전할 문장을 다듬는 장면
- 효과: 기록과 전달 루프 강화
- 신규 배경 추천

#### 6. `market_people`

- 위치: `downtown` 이후
- 역할: 상인, 주민, 학생이 섞인 현장 대화
- 효과: 항쟁이 시민 전체의 일로 넓어지는 감각 강화
- 신규 배경 추천

#### 7. `citizen_debate`

- 위치: `citizen_voice` 직후
- 역할: 남을지, 알릴지, 도울지 시민들이 함께 토론
- 효과: 선택의 무게 강화
- 기존 `plaza` 재사용 가능

### C. 후반 확장

#### 8. `supply_run`

- 위치: `help_people`와 `community` 사이
- 역할: 물, 음식, 소식 전달
- 효과: 공동체 유지 과정을 플레이로 체감
- 기존 `square` 재사용 가능

#### 9. `checkpoint_edge`

- 위치: `outside_message` 진입 전
- 역할: 외곽 검문, 우회 여부, 기록 전달 긴장감
- 효과: 후반 긴장 강화
- 신규 배경 추천

#### 10. `night_meeting`

- 위치: `community`와 `last_night` 사이
- 역할: 남을 사람, 떠날 사람, 기록을 맡을 사람의 마지막 논의
- 효과: 엔딩 직전 감정 축적
- 신규 배경 추천

## 구현 우선순위

### 1차: 배경 추가 없이 가능한 확장

- `station_rumor`
- `family_neighborhood`
- `citizen_debate`
- `supply_run`

이 4개만 넣어도 체감 분량이 꽤 늘어난다.

### 2차: 신규 배경이 있으면 좋은 확장

- `side_alley_detour`
- `street_clinic`
- `leaflet_room`
- `market_people`
- `checkpoint_edge`
- `night_meeting`

## 새 씬 추가 시 문장 규칙

- UI 라벨은 명사형으로 짧게 유지한다.
- 장면 본문은 주인공의 현장 감각이 자연스럽게 느껴지게 쓴다.
- `나는`을 문장마다 붙이지 않는다.
- 역사 설명은 `history`에서만 해설형으로 쓴다.
- 선택지 설명은 "무슨 감정"보다 "무슨 행동"인지 먼저 드러낸다.
