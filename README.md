# 오월, 그 거리에서

> 1980년 5월 광주, 평범한 시민의 눈으로 바라본 인터랙티브 픽셀 게임

5·18 민주화운동을 주제로 한 텍스트 어드벤처 웹 게임입니다.  
플레이어의 선택에 따라 이야기가 분기되며, 광주 곳곳을 돌아다니며 그날의 기억을 따라갑니다.

## 기술 스택

- **프레임워크** — Next.js (App Router), React 19, TypeScript
- **스타일** — Tailwind CSS v4, 픽셀 폰트 (Press Start 2P)
- **DB** — Neon (PostgreSQL) — 플레이 기록 저장

## 실행

```bash
npm install
npm run dev
```

## 주요 기능

- 분기형 스토리 (선택지마다 결말이 달라짐)
- 실제 광주 지리 기반 활동 지도 (미니맵 + 광주 전도)
- 누적 기록 — 여러 회차에 걸친 방문 장소·선택 이력 보존
- 픽셀 아트 씬 일러스트
